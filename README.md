# Step Sequencer

An app that uses Tone.js to create a step sequencer.

**Programming Language**: JavaScript

**Web Framework**: React

**Library**: [Tone.js](https://tonejs.github.io/)

**Markup/Styling**: HTML5/SCSS

The sequencer can:
- increase/decrease BPM
- increase/decrease swing
- increase/decrease delay
- increase/decrease gain
- increase/decrease octave
- Select from a range of synths
- Record and download audio

---

# Setup

- Download the files

- Navigate to project folder and run:

```
npm i
```
```
npm start
```
